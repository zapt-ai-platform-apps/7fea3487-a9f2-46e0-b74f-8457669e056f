# Situational Awareness Tool

A web application for emergency and police services to identify risks and threats from social network information and integrate with mapping services.

## Features

- Monitor social media platforms (Twitter/X, Instagram, TikTok) for potential threats
- Visualize threats and incidents on Google Maps
- Analyze social media content for risk assessment
- Track threat levels and generate reports
- Configure monitoring settings and alert thresholds

## Technologies Used

- React.js
- Google Maps API
- Social Media APIs (Twitter, Instagram, TikTok)
- Tailwind CSS

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Add the required environment variables in `.env` file
4. Start the development server: `npm run dev`

## Testing

Run tests with: `npm test`
